# Extracted boot assets

Read-only **reference copies** of what the machine shows at boot. Regenerated by `bin/themes-extract`.

| File | Size | Source | Role on screen |
|------|------|--------|----------------|
| `bgrt-firmware-oem.bmp` | ~274 KB | `/sys/firmware/acpi/bgrt/image` | **Center Dell logo** (UEFI BGRT, 304×307 BMP) |
| `bgrt-firmware-oem.png` | ~18 KB | converted from BMP | Preview / edit in GIMP, then use as `--oem` background |
| `ubuntu-watermark-dark.png` | 187×72 | `/usr/share/pixmaps/ubuntu-logo-text-dark.png` | **Bottom “Ubuntu” text** on dark splash |
| `ubuntu-watermark-light.png` | 187×72 | `/usr/share/pixmaps/ubuntu-logo-text.png` | Light-variant watermark (stock theme uses dark) |

## Modify

- **Do not edit files here** — they are overwritten by `bin/themes-extract`.
- Copy assets to `../overlay/` and run `sudo bin/themes-install-boot`.
- To change the Dell center logo permanently in firmware: Dell BIOS setup (outside Plymouth).

## Technical notes

Plymouth `two-step` module reads:

- Firmware image: `/sys/firmware/acpi/bgrt/image` when `UseFirmwareBackground=true`
- Fallback: `spinner/bgrt-fallback.png` if BGRT missing
- Watermark: `watermark.png` in theme dir (symlinked to ubuntu pixmap in stock spinner)
